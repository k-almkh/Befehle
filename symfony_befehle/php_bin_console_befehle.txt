Usage:
  command [options] [arguments]

Options:  
php bin/console -h, --help            Display help for the given command. When no command is given display help for the list command
php bin/console -q, --quiet           Do not output any message
php bin/console -V, --version         Display this application version
php bin/console --ansi|--no-ansi  Force (or disable --no-ansi) ANSI output
php bin/console -n, --no-interaction  Do not ask any interactive question
php bin/console -e, --env=ENV         The Environment name. [default: "dev"]
php bin/console --no-debug        Switch off debug mode.
php bin/console -v|vv|vvv, --verbose  Increase the verbosity of messages: 1 for normal output, 2 for more verbose output and 3 for debug
