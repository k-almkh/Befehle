((((((((( php8 )))))))))))
alte PHP löschen:
	sudo apt purge '^php7.4.*'
	sudo apt update
Aktivieren des PHP-Repositorys
	sudo apt install software-properties-common
	sudo add-apt-repository ppa:ondrej/php
	
PHP als Apache-Modul installieren
	sudo apt update
	sudo apt install php8.0 libapache2-mod-php8.0
	sudo systemctl restart apache2
	php -v
Apache mit PHP-FPM konfigurieren
	sudo apt update
	sudo apt install php8.0-fpm libapache2-mod-fcgid
Standardmäßig ist PHP-FPM in Apache nicht aktiviert. Um es zu aktivieren, führen Sie Folgendes aus:
	sudo a2enmod proxy_fcgi setenvif
	sudo a2enconf php8.0-fpm
	sudo systemctl restart apache2
	
Installieren Sie PHP 8-Erweiterungen
	sudo apt install libapache2-mod-php8.0 php8.0-zip php8.0-xml php8.0-mbstring php8.0-gd php8.0-curl php8.0-imagick php8.0-intl php8.0-bcmath php8.0-gmp php8.0-cli php8.0-mysql php8.0-pgsql php8.0-ldap php8.0-zip php8.0-gd  php8.0-mbstring php8.0-curl php8.0-xml php8.0-apcu php8.0-redis
	sudo systemctl restart apache2
	php -m
php.ini ändern:
	sudo nano /etc/php/8.0/apache2/php.ini
php version ändern: 
	sudo update-alternatives --set php /usr/bin/php7.4
	sudo apt install libapache2-mod-php7.4 php7.4-zip php7.4-xml php7.4-mbstring php7.4-gd php7.4-curl php7.4-imagick php7.4-intl php7.4-bcmath php7.4-gmp php7.4-cli php7.4-mysql php7.4-pgsql php7.4-ldap php7.4-zip php7.4-gd  php7.4-mbstring php7.4-curl php7.4-xml php7.4-apcu php7.4-redis
	sudo systemctl restart apache2
configure Apache to use PHP 8.0:	
	a2enmod php8.0
	
PHP 7.4	
	sudo apt -y install php7.4
	sudo apt-get install -y php7.4-{bcmath,bz2,intl,gd,mbstring,mysql,zip,common, }
